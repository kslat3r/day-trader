> Scrapes history stock market data


## Install

```sh
$ npm install --save data-scraper
```


## Usage

```js
var dataScraper = require('data-scraper');

dataScraper('Rainbow');
```

```sh
$ npm install --global data-scraper
$ data-scraper --help
```
